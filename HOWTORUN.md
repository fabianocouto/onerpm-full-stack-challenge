# ONErpm Full Stack Challenge

## Dependencies

- Docker
- Docker Compose
- [Installation Help](https://www.nerdlivre.com.br/instalando-docker-e-docker-compose-no-ubuntu-24-04/)

## Run application

At the root directory, run the follow command:

```
$ docker compose up
```
